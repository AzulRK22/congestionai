# CongestionAI — Web

**Qué es:** Origen/Destino → **Planear** → “Sal a las HH:MM”, ahorro estimado, .ics y WhatsApp.  
**Stack:** Next.js (App Router) + TypeScript + Tailwind + APIs mock.

## Correr
```bash
pnpm i
pnpm dev
```

## Scripts
- `pnpm dev` — modo desarrollo
- `pnpm build` — compila producción
- `pnpm start` — arranca servidor

## Notas
- Modelo horario promedio (168h). Los eventos locales pueden variar.
